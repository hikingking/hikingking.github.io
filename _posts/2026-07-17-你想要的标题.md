  # **一步步添加新博客文章**
  
  ## 步骤 1：创建新文章文件
  访问这个链接创建新文件：
  
  Code
  https://github.com/hikingking/hikingking.github.io/new/master/_posts
  ## 步骤 2：填写文件名
在 "Name your file..." 输入框中输入：

Code
2026-07-17-你想要的标题.md
  格式说明：
  
  2026-07-17 是日期（YYYY-MM-DD 格式）
  - 是连字符
  你想要的标题 是文章标题（可以用中文或英文）
  .md 是 Markdown 文件扩展名
  例子：
  
  2026-07-18-我的第二篇文章.md
  2026-07-18-How-to-write-blog.md
  ## 步骤 3：编写文章内容
  在编辑框中输入你的文章内容（Markdown 格式）：
  
  Markdown
  # 文章标题
  
  这是文章的内容。可以用 Markdown 格式编写。
  
  ## 副标题
  
  - 列表项 1
  - 列表项 2

**加粗文本** 和 *斜体文本*
    ## 步骤 4：提交文章
  滚动到页面底部，填写提交信息：
  
  Commit message：输入 创建新博客文章 或其他描述
  点击绿色按钮 "Commit new file"
  ✅ 文章已创建！

  ## 步骤 5：更新侧边栏导航
现在需要在菜单中添加这篇文章的链接。

访问这个链接编辑导航文件：

Code
https://github.com/hikingking/hikingking.github.io/edit/master/_sidebar.md
在打开的编辑器中，找到这些内容：

Markdown
* [首页](#/)
* [下载安装](download.md)
* [使用指南](guide.md)
* [第一篇](_posts/2026-07-17-第一篇.md)
* [关于](about.md)
在合适位置添加新一行（例如在"第一篇"下面）：

Markdown
* [你文章的标题](_posts/2026-07-17-你想要的标题.md)
完整示例：

Markdown
* [首页](#/)
* [下载安装](download.md)
* [使用指南](guide.md)
* [第一篇](_posts/2026-07-17-第一篇.md)
* [我的第二篇文章](_posts/2026-07-18-我的第二篇文章.md)
* [关于](about.md)
步骤 6：提交导航修改
滚动到页面底部
Commit message：输入 更新导航，添加新文章链接
点击 "Commit changes"
✅ 完成！
